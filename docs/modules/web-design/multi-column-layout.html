CSS3 Create Multiple Columns

The column-count property specifies the number of columns an element should be divided into

Example

View Example 
<!DOCTYPE html>
<html>
<head>
    <style>
    .newspaper {
        column-count: 3;
    }
    </style>
</head>

<body>
    <p>
        <b>Note:</b> Internet Explorer 9, and earlier versions, does not support the column-count property.
    </p>
